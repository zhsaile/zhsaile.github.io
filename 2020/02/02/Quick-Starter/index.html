Quick Starter

Posted on 2020-02-02 8.6k
A quick start of Qt Quick

QML Syntax

   QML界面设计时遵循元素继承的原则，父元素可以有多个子元素，子元素继承父元素的坐标系统。部件的坐标 (x, y) 均是相对父元素而言的。在进行界面设计时，建议对每一个元素都进行命名，使其拥有一个独一无二的id，这样便可以对任意一个元素进行索引、操作。另外建议将根元素命名为root以便引用。

   在QML文件的头部需要引用QtQuick库，其版本与主机所安装的Qt版本对应。比如Qt 5.12对应QtQuick 2.12，以此类推。

Properties

    部件元素自身便具有一些属性，比如最常见的width, height, x, y等，另外用户还可以自定义一些属性，通过使用alias关键字可以设置自定义属性的别称，这会在后面从自定义的部件中导出内部属性时用到，设置别称时不需要再定义type，与所引用的变量的type相同。

For every property, you can provide a signal handler. This handler is called after the proterty changes. 例如部件高度变化的 handler 为onHeightChanged，自定义property的handler为onPropertyChanged.

在QML中定义的变量，若发生了改变，那么变量所在的部件会自动更新。

Scripting

  QML and JavaScript (also known as ECMAScript) are best friends : ) The JavaScript function can access the properties in QML and can be easily invoked by the QML codes .

    自定义property的自动更新是建立在与 key 之间存在 binding 的基础之上的，binding is a contract and keeps true over the lifetime of the binding，例如Text部件的text属性：text: "Hello " + name，如果name发生了变化，那么Text部件的显示便会自动更新。但是，当二者之间是assignment的关系时，便会打破之前的binding, the value will not update the display anymore as the previous binding of the property was destroyed. It is a kind of broken contract!

Basic Elements

  Qt Quick 中的元素可以被分为可视元素和不可视元素两类。Rectangle, Text 是两个典型的可视元素，通常都会显示在屏幕上。Timer 是一个不可视元素，通常用来操作可视元素。

Item Element

  Item is the base element for all visual elements as such all other visual elements inherits from it. It doesn’t paint anything by itself but defines all properties which are common across all visual elements:

Rectangle Element

  The Rectangle extends Item and adds a fill color to it. Additionaly it supports borders defined by border.color and border.width. To create rounded rectangles you can use the radius property.

Text Element

  The text in Text element can be aligned to each side (Text.AlignTop, Text.AlignBottom, Text.AlignLeft, Text.AlignRight, Text.AlignHCenter, Text.AlignVCenter) and the center using the horizontalAlignment and verticalAlignment properties.

  A Text element only displays the given text and it doesn’t render any background decoration. Besides the redered text the Text is transparent. In addition, a Text initial width or height is depending on the text string. A Text with no widths set and no text will not be visible.

  在对Text进行布局的时候，要分清楚是对Text中的文字进行布局还是对Text部件本身进行布局。如果是前者，则对horizontalAlignment和vertivalAlignment进行设置，如果是后者，则对anchors进行设置。

MouseArea Element

  通过定义一个MouseArea便可以接收鼠标的点击信号，实现对 Element 的各种操作。这是 QtQuick 的一个显著的特性，利用这个特性可以定义一个与显示界面大小不同的操作界面。

Components

  A component is a reusable element and QML provides different ways to create components. The simplest way is creating a file-based component and giving the file an element name (e.g. Button.qml). The component can be used like other element from the QtQuick module.

Simple Transformation

  QML Items can, in general, be translated, rotated and scaled. There is a simple form of these operations and a more advanced way.

Layout Items

  In addition to the more traditional Grid, Row and Column, Qt Quick provides a way to layout items using the concept of anchors. Each item can be thought of as having a set of 7 visible “anchor lines”: left, horizontalCenter, right, top, verticalCenter, baseline, bottom.

anchors.fill is the same as setting the left, right, top and bottom anchors to the left, right, top and bottom of the target item. The width and height will be changed in such conditions. 在这种情况下，元素的尺寸（width height）将会发生变化并自动适应目标元素。

anchors.centerIn is the same as setting the verticalCenter and horizontalCenter anchors to the verticalCenter and horizontalCenter of the target item.
