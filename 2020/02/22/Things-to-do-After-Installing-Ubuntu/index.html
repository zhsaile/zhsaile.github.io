Vim

Update Vim

Update vim to vim8:

Install vim plugin manager

At that time the popular vim plugin manager is vim-plug whose github page is vim-plug.

Adding the plugins name in .vimrc.
Run :PlugInstall to install plugins.

Deleting or commenting the plugins for the plugings you want to remove
Run :PlugClean to remove the plugins and related folders.

Run :PlugUpdate to update the plugins.

Install LeaderF

LeaderF is an effective fuzzy finder to search files in vim.

Install tabular & vim-markdown

Certain features like folding in the vim-markdown have dependency on the plugin tabular thus the plugin should be installed boundly. Besides, the tabular plugin must come before vim-markdown.

Install vim-qml

Support qml syntax highlight.
