Posted on 2020-02-22 5k

 

Vim

Update Vim

Update vim to vim8:

Install vim plugin manager

At that time the popular vim plugin manager is vim-plug whose github page is vim-plug.

Adding the plugins name in .vimrc.
Run :PlugInstall to install plugins.

Deleting or commenting the plugins for the plugings you want to remove
Run :PlugClean to remove the plugins and related folders.

Run :PlugUpdate to update the plugins.

Install LeaderF

LeaderF is an effective fuzzy finder to search files in vim.

Install tabular & vim-markdown

Certain features like folding in the vim-markdown have dependency on the plugin tabular thus the plugin should be installed boundly. Besides, the tabular plugin must come before vim-markdown.

Install vim-qml

Support qml syntax highlight.

bash

autojump

Open at specified position

  1. 设置按下ctrl alt T时在特定的物理位置出现, 首先将终端放在屏幕中想要放置的位置
  2. 运行xwininfo, 此时鼠标变成十字准星星, 点击终端窗口, 此时便会显示终端的相关信息, 记录其中的-geometry信息
  3. 打开系统的快捷键设置, 将默认的终端快捷键设置成别的, 比如ctrl shift alt T, 在最后添加快捷键, command为gnome-terminal --geometry=80*24--10--10, 其中 geometry 即为 xwininfo 记录的位置几何信息, 将该命令的快捷键设置为 ctrl alt T 即可.

grub

2020年三月双系统安装记录:

  1. 在nvme上安装了ubuntu18.04, 然后在sata ssd上面安装了windows10 1909.
  2. 启动后会直接进入windows, 并且在bios中找不到ubuntu的启动项.
  3. 安装easyuefi, 检测后发现ubuntu的启动项处在隐藏状态
  4. 由于没有查到如何使隐藏的启动项变为启用, 因此新建一个启动项, 选择为 ubuntu 系统, 启动文件可以在前面隐藏的那个启动项的详细信息下找到, 将新建立的启动项移动至首位
  5. 重新启动, 便会直接进入ubuntu
  6. 再次重启后发现bios启动项中windows的启动项又消失了, 进入ubuntu, 命令行中输入 sudo update-grub, 重启按F12, 便会看到选择系统的界面.

samba

在ios11版本苹果终于加入了文件管理系统, 不仅可以管理照片/文档, 还支持其它云存储软件比如Google Drive, OneDrive, Dropbox等. 另外还支持通过samba协议连接其他设备, 实现与其他设备的文件共享, 从此远离数据线, 再也不用使用wechat或者邮件来传递文件了, 最最最优秀的是由于是使用局域网传输, 传输速度可以达到非常高.

Windows

  • 任意位置建立一个共享文件夹share, 右键–>属性
  • 点击共享 --> 高级共享 --> 勾选共享此文件夹✔️
  • 点击权限, Everyone的权限全部勾选允许✔️, 点击确定
  • 回到共享标签, -->共享(s), 下拉菜单选择Everyone, 点击添加, 权限改为读取/写入, 点击确定
  • win+R输入cmd打开命令行, 记录用户名user; 输入ipconfig, 记下电脑的ipv4地址
  • 打开ios自带的文件APP, 点击右上角三个小点, 选择连接服务器(connect to server), 服务器输入 smb://ipv4地址, 连接.
  • 选择注册用户, Name为电脑User用户名, 密码为微软帐号密码或者是电脑登录密码, 连接,大功告成.
