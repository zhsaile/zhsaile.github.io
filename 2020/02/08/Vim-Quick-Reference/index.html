Vim Quick Reference

Posted on 2020-02-08 281
Vim 中容易忘记却有用的技巧

删除

命令 功能
D 删除至行尾
C 删除至行尾并插入
DW 删除到某个以空格作为分隔符的单词的结尾位置
CW 删除到某个以空格作为分隔符的单词的结尾位置并插入
d) 删除到某个语句的结尾位置
c) 删除到某个语句的结尾位置并插入
dfc 删除到字符 c 出现的位置并删除 c
dtc 删除到字符 c 出现的位置并删除 c，然后插入
cfc 删除到字符 c 出现的位置并保留 c
ctc 删除到字符 c 出现的位置并保留 c，然后插入

跳转

命令 功能
% 跳转至匹配的括号处

分屏

命令 功能
:sp [FILENAME] 水平分屏打开文件
:vs [FILENAME] 垂直分屏打开文件
CTRL W + 扩大窗口
CTRL W - 缩小窗口
